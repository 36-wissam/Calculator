## Project Structure

- `index.html` – Calculator structure and buttons layout
- `style.css` – Styling, layout, calculator design, and custom fonts
- `script.js` – Calculator logic and button functionality

---

## What I Learned

- How to handle calculator buttons and user input values using JavaScript
- How to connect buttons with functions using `eventListener`
- How to work with values and perform math operations
- How to use `parseFloat()` to convert strings into numbers
- How to handle common calculator errors like:
  - Division by zero
  - Pressing operation buttons multiple times
- Understanding more about the `DOM`
- How to import fonts from Google Fonts and use them in CSS
- Learned more CSS techniques for improving the interface design
